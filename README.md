### Blenderset
This Code is lazypic blender setting code.

#### install
```
cd ~ && git clone https://github.com/lazypic/blender.git
```

#### how to run blender
```
// macOS : ~/.profile
echo "alias blender=\"/Applications/Blender/blender.app/Contents/MacOS/blender --python ~/blender/init.py\"" >> ~/.profile

// linux
```
